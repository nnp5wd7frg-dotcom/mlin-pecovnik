# Mlin Pečovnik

Next.js 15 + React + TypeScript + Tailwind CSS spletna stran za Mlin Pečovnik.

## Zagon

```bash
npm install
npm run dev
```

Odpri: http://localhost:3000

## Podatki podjetja

- Mlin Pečovnik
- Dejan Pečovnik s.p.
- Mlinska ulica 28, 2313 Fram, Slovenija

## Opomba

Projekt je izvirna spletna predstavitev, vizualno navdahnjena z moderno korporativno mlinarsko spletno stranjo. Ne vsebuje tujih logotipov, zaščitenih besedil ali slik.
